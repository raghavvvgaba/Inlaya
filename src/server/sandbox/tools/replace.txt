Replace exact text on one inspected line.

Usage:
- Use this for targeted edits after reading the relevant lines.
- `path` should be a repository-relative file path.
- `startLine` is 1-based and should come from read_file output.
- `oldText` must be exact text from the target line.
- The target line must contain `oldText` exactly once.
- Use write_file instead for full rewrites or new complete file contents.
